# House Price Prediction using Machine Learning

##  Problem Statement
Predict house prices based on different features using machine learning models.

##  Dataset
California Housing Dataset

##  Models Used
- Linear Regression
- Random Forest Regressor

##  Results
- Linear Regression R²: 0.57
- Random Forest R²: 0.80

##  Best Model
Random Forest performed best due to ability to capture non-linear relationships.

## Key Insight
Median Income is the most important feature affecting house prices.

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
